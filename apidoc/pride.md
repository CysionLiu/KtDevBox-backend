
>本页面包含2个接口，分别为点赞博客，取消点赞

主域名：http://148.70.2.189/app/


#### 1.点赞博客

**路径**：主域名+blog/pride

**方法**：POST

**Header**：需要

**参数**：


字段  | 说明 | 是否必传
---|---|---
blogId | 博客id|是

**返回值示例**：
```
{
    "code": 200,
    "msg": "success"
}
```

---


#### 2.取消点赞

**路径**：主域名+blog/unpride

**方法**：POST

**Header**：需要

**参数**：


字段  | 说明 | 是否必传
---|---|---
blogId | 博客id|是

**返回值示例**：
```
{
    "code": 200,
    "msg": "success"
}
```
